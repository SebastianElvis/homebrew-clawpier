# homebrew-clawpier

Homebrew tap for [ClawPier](https://github.com/SebastianElvis/clawpier) — a macOS desktop app for managing Dockerized OpenClaw bot instances.

## Install

```bash
brew tap SebastianElvis/clawpier
brew install --cask clawpier
```

## Update

```bash
brew upgrade --cask clawpier
```

## Uninstall

```bash
brew uninstall --cask clawpier
```
